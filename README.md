# Quantum Sudoku

A game build in Nuxt exploring a "quantum" take on Sudoku. This is a work-in-progress and everything is subject to changes.

## Tech Stack

- Nuxt 4 + Vue 3
- `@nuxtjs/color-mode`, `@nuxt/fonts`, `@nuxt/icon`, `@nuxt/image`
- ESLint via `@nuxt/eslint`

## Getting Started

### 1) Install dependencies

```bash
npm install
```

### 2) Run development server

```bash
npm run dev
```

App runs at `http://localhost:3000` by default.

## Available Scripts

```bash
npm run dev       # Start Nuxt dev server
npm run build     # Build for production
npm run preview   # Preview production build locally
npm run generate  # Generate static output
```
